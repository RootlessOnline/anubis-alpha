## Processing Stages (Plain English)
1. INPUT - Parse message
2. MEMORY - Search related memories
3. LOGIC - Sentiment analysis
4. EMOTION - Detect feelings
5. JUDGMENT - Truth vs comfort
6. COMPASSION - Empathy needs
7. BALANCE - Harmonize
8. UNDERSTANDING - Pattern recognition
9. INSIGHT - Generate intuition
10. SYNTHESIS - Combine all
11. INTENT - Set response intention

## Memory Tiers
- **Working Memory:** 4 slots, 60s decay
- **Long-term Memory:** 10k capacity, persistent
- **Core Memory:** Permanent identity

## State System
- Energy (0-100) with processing costs
- IQ/EQ (0-100) grow with memory creation
- Mood system with 8 states
